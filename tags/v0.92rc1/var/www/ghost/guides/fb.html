Forced Browsing

Forced Browsing is a very simple technique commonly called dot slash. This type of access is capable by mis-configured servers that allow file searching through the browser. Here is a quick example:
original address - http://www.mysite.com/index.php
Forced Browse - http://www.mysite.com/../../../../../the file you want to access/
URL Encoded - http://www.mysite.com/..%2f..%2f..%2f..%2f..%2fthe file you want to access/
