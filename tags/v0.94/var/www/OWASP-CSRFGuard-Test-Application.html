OWASP CSRFGuard Test Application

The links below are to two versions of the test application that comes with the OWASP CSRFGuard.

  • OWASP CSRFGuard Test Application (Protected) - Version of the CSRFGuard Test Application with the CSRFGuard turned on. Should not contain any vulnerabilities.
  • OWASP CSRFGuard Test Application (Vulnerable) - Version of the CSRFGuard Test Application with the CSRFGuard turned off.
